# ТЕХНИЧЕСКОЕ ЗАДАНИЕ  
для проекта «Фронтенд и бэкенд-разработка многофункционального приложения для управления личным портфолио»

## 1. ВВЕДЕНИЕ

Наименование работы: **Фронтенд и бэкенд-разработка многофункционального приложения для управления личным портфолио**.  
Продуктом работы является сайт-портфолио с расширяемым функционалом, на котором можно ознакомиться с правилами настольной ролевой игры Shadowrun, составить список задач, выполнять скрипты на Golang и хранить файлы.

## 2. ОСНОВАНИЕ ДЛЯ РАЗРАБОТКИ

Дата утверждения: **не указана**.  
Тема разработки: **portfolio-management-system v1.0**.

## 3. НАЗНАЧЕНИЕ РАЗРАБОТКИ

**Функциональное назначение:**  
Создание комплексной платформы для управления личным портфолио, включая справочную систему по настольным играм, планировщик задач, облачное хранилище файлов и песочницу для выполнения Golang скриптов.

**Эксплуатационное назначение:**  
Использование в личных и учебных целях для организации информации, управления задачами и хранения файлов в единой системе.

## 3. Описание логической структуры

**Алгоритм программы:**
Пользователь входит в систему → система проверяет авторизацию → предоставление доступа к модулям:
- Справочник Shadowrun: поиск, категории, теги
- Задачи: создание, редактирование, фильтрация, папки
- Хранилище: загрузка, скачивание, управление файлами
- Скрипты: выполнение и сохранение Golang кода

**Используемые методы:**
- JWT для аутентификации и авторизации
- GORM для работы с базой данных
- REST API архитектура
- bcrypt для хеширования паролей
- Middleware для контроля доступа

**Структура программы:**
```
portfolio/
├── frontend/                   # Фронтенд
│   ├── assets/                 # ассеты
│   ├── css/                    # Стили
│   ├── html/                   # HTML страницы
│   │   ├── index.html          # Главная
│   │   └── page/               # Страницы модулей
│   └── javascript/             # Логика фронтенда
└── backend/go/                 # Бэкенд на Go
    ├── main.go                 # Точка входа
    ├── database/               # Работа с БД
    ├── models/                 # Структуры данных
    ├── handlers/               # Обработчики API
    └── middleware/             # Middleware
```

**Основные модули:**
- `AuthHandler` - аутентификация и авторизация
- `TaskHandler` - управление задачами
- `FileHandler` - работа с файлами
- `ScriptHandler` - выполнение скриптов
- `ShadowrunHandler` - справочная система

**Связи программы с другими программами:**
- Интеграция с базой данных PostgreSQL
- Веб-сервер Gin
- API для взаимодействия с фронтендом

### Примеры использования
**Запуск в среде разработки:**

1. Установите PostgreSQL и создайте базу данных 'portfolio_db'
2. Выполните SQL скрипт инициализации из `backend/go/database/init.sql`
3. Настройте .env файл:
```
DB_HOST=localhost
DB_PORT=5432
DB_USER=kayah
DB_PASSWORD=81
DB_NAME=portfolio_db
PORT=8080
JWT_SECRET=your_secret_key
```
4. Запустите приложение:
```bash
cd backend/go
go run main.go
```

**Доступ к приложению:**
1. Откройте браузер и перейдите по адресу http://localhost:8080
2. Войдите используя тестовые учетные записи:
   - Логин: `admin`, Пароль: `admin123`
   - Логин: `user`, Пароль: `user123`
3. Получите доступ ко всем модулям системы
